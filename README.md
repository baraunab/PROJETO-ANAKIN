## Funcionalidades 🎮
**ANAKIN** permite que o Mestre da mesa controle sua sessão de RPG com registros de fichas de personagens, campos de anotações e histórico de rolagem de dados.

## Desenvolvimento e Ferramentas (Windows 11)
- **Linguagens**: Java 21 e SQL 2022
- Eclipse IDE
- MySQL Workbench
- JDBC Driver for MySQL (Connector/J) - [Plataform Independent](https://dev.mysql.com/downloads/connector/j/)

### Configuração ⚙️
No Eclipse IDE, adicione o arquivo **jar** nas Propriedades do projeto:
```
Properties > Java Build Path > Libraries > Add External JARs
```
